#  Pizza Sales Data Analysis using SQL

This project analyzes Pizza Hut sales data using SQL Server.

The dataset was imported from 4 CSV files into a relational database, and multiple analytical queries were written to extract business insights.


# Dataset

The project uses 4 CSV files:

- orders.csv
- order_details.csv
- pizzas.csv
- pizza_types.csv

These files were imported into SQL Server and structured into relational tables.


# Tools Used

- SQL Server Management Studio (SSMS)
- CSV Data Import


# Key Business Questions Solved

# Basic Analysis
- Total number of orders
- Total revenue generated
- Highest priced pizza
- Most common pizza size

# Intermediate Analysis
- Top 5 most ordered pizza types
- Category-wise total quantity
- Orders distribution by hour
- Average pizzas ordered per day

# Advanced Analysis
- Top 3 pizza types by revenue
- Percentage contribution of each pizza to total revenue
- Cumulative revenue over time (Window Functions)
- Top 3 pizzas per category using CTE & RANK()


##  Advanced SQL Concepts Used

- JOIN (Multiple table joins)
- GROUP BY & HAVING
- Aggregate Functions (SUM, COUNT, AVG)
- Window Functions (DENSE_RANK, RANK, OVER)
- CTE (Common Table Expression)
- DATEPART & Date Casting
- Revenue calculations
- Ranking within categories


# Key Insights Generated

- Identified highest revenue generating pizzas
- Found peak order hours
- Analyzed category-wise performance
- Measured daily and cumulative revenue trends
- Calculated revenue contribution percentage per pizza

The goal of this project was to:

- Strengthen SQL analytical skills
